## Полезные команды

```gcode
QUERY_PROBE                          ; проверить состояние Z датчика
ACCELEROMETER_QUERY                  ; тест ADXL345
QUERY_ADC NAME=heater_bed            ; напряжение на термисторе стола

G28 X Y                              ; хоминг только XY
G28                                  ; полный хоминг XYZ (нужен probe!)
QUAD_GANTRY_LEVEL                    ; выравнивание портала

SET_KINEMATIC_POSITION Z=0           ; установить Z=0 без хоминга (force_move)
G91                                  ; относительные координаты
G1 Z10 F300                          ; поднять Z на 10 мм

PID_CALIBRATE HEATER=heater_bed TARGET=60
PID_CALIBRATE HEATER=extruder TARGET=220
PROBE_CALIBRATE
SHAPER_CALIBRATE
SAVE_CONFIG                          ; записать калибровки в printer.cfg
```
